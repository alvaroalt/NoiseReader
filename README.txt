  1.  ¿QUÉ ES NOISE READER?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Noise Reader es una aplicación de lectura que corre directamente
  en tu navegador web (Chrome, Firefox, Edge o Safari).

  No necesitás instalar nada.
  No necesitás crear una cuenta.
  No sube tus archivos a ningún servidor.
  Todo sucede en tu propia computadora.

  La idea principal es que podés cargar un texto o un PDF y leerlo
  sobre una capa de ruido visual animado que podés ajustar a tu
  gusto. También incluye herramientas opcionales para mejorar la
  velocidad de lectura.

  ┌─────────────────────────────────────────────────────────────┐
  │  Pensalo como un libro con una pantalla que "respira".      │
  │  El ruido de fondo le da a la lectura una textura más       │
  │  interesante y, para algunas personas, ayuda a mantener     │
  │  la concentración.                                          │
  └─────────────────────────────────────────────────────────────┘

  2.  LO QUE NECESITÁS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Para usar Noise Reader no necesitás saber nada de programación.
  Solo tres cosas:

    ·  Un navegador web moderno
       Google Chrome, Mozilla Firefox, Microsoft Edge o Safari.
       Cualquiera de estos sirve.

    ·  El archivo  noise-reader.html
       Es el único archivo de la aplicación.

    ·  Un archivo .txt o .pdf que quieras leer.

  ┌─────────────────────────────────────────────────────────────┐
  │  ¿No sabés si tu navegador es moderno? Si lo actualizaste   │
  │  en los últimos dos años, muy probablemente sí lo es.       │
  │  Si tenés dudas, descargá Chrome desde:                     │
  │  www.google.com/chrome                                      │
  └─────────────────────────────────────────────────────────────┘

  Seguí estos pasos, uno por uno:

  ┌───┬──────────────────────────────────────────────────────────┐
  │ 1 │  ENCONTRÁ EL ARCHIVO                                     │
  │   │  Buscá  noise-reader.html  en tu computadora.            │
  │   │  Puede estar en la carpeta de Descargas o donde          │
  │   │  lo hayas guardado.                                      │
  ├───┼──────────────────────────────────────────────────────────┤
  │ 2 │  ABRILO CON DOBLE CLIC                                   │
  │   │  En la mayoría de las computadoras, esto lo abre         │
  │   │  automáticamente en tu navegador web.                    │
  ├───┼──────────────────────────────────────────────────────────┤
  │ 3 │  SI NO SE ABRE SOLO...                                   │
  │   │  Clic derecho sobre el archivo                           │
  │   │  → "Abrir con"                                           │

  ├───┼──────────────────────────────────────────────────────────┤
  │ 4 │  ¡LISTO!                                                 │
  │   │  Deberías ver la pantalla con el fondo oscuro animado    │
  │   │  y la barra de controles en la parte superior.           │
  └───┴──────────────────────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  La aplicación no necesita internet para funcionar una      │
  │  vez que tenés el archivo. Podés llevarla en un pendrive    │
  │  y usarla en cualquier computadora.                         │
  └─────────────────────────────────────────────────────────────┘

  4.  CÓMO CARGAR UN ARCHIVO PARA LEER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Hacé clic en el botón  [ ↑ CARGAR ARCHIVO ]  que aparece en la
  esquina superior izquierda de la pantalla.

  Se va a abrir el explorador de archivos de tu sistema. Desde ahí
  navegá hasta el archivo que querés leer y seleccionalo.

  La aplicación acepta dos tipos:

  ┌──────────────────┬───────────────────────────────────────────┐
  │  Archivo .txt    │  Texto plano sin formato.                 │
  │                  │  Compatible con todas las funciones,      │
  │                  │  incluyendo Lectura Biónica.              │
  ├──────────────────┼───────────────────────────────────────────┤
  │  Archivo .pdf    │  Documentos con texto e imágenes.         │
  │                  │  El texto se extrae y las imágenes        │
  │                  │  aparecen intercaladas. También es        │
  │                  │  compatible con Lectura Biónica.          │
  └──────────────────┴───────────────────────────────────────────┘

  ┌─────────────────────────────────────────────────────────────┐
  │  Si el texto no se ve bien o aparecen caracteres extraños,  │
  │  puede ser que el archivo tenga una codificación especial.  │
  │  Intentá guardarlo nuevamente desde el editor de texto      │
  │  eligiendo la opción  UTF-8.                                │
  └─────────────────────────────────────────────────────────────┘

  5.  LOS CONTROLES DE LA APLICACIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  En la barra superior vas a encontrar todos los controles:

  ┌────────────────────┬─────────────────────────────────────────┐
  │  [ ↑ CARGAR ]      │  Abre el explorador de archivos para    │
  │                    │  elegir el .txt o .pdf a leer.          │
  ├────────────────────┼─────────────────────────────────────────┤
  │  RUIDO  [──●────]  │  Controla la intensidad del ruido       │
  │                    │  animado. Deslizá a la derecha para      │
  │                    │  más ruido, a la izquierda para menos.  │
  ├────────────────────┼─────────────────────────────────────────┤
  │  [ ESTÁTICO ]      │  Ruido clásico tipo "TV sin señal".     │
  │  [ SCAN    ]       │  Una línea luminosa que barre la        │
  │                    │  pantalla de arriba a abajo.            │
  │  [ GRANO   ]       │  Ruido suave de película analógica.     │
  ├────────────────────┼─────────────────────────────────────────┤
  │  [ ◑ INVERTIR ]    │  Alterna entre fondo oscuro con texto   │
  │                    │  claro (modo noche) y fondo claro con   │
  │                    │  texto oscuro (modo día).               │
  ├────────────────────┼─────────────────────────────────────────┤
  │  [ B| BIÓNICO ]    │  Activa la Lectura Biónica. Las         │
  │                    │  primeras letras de cada palabra se      │
  │                    │  ponen en negrita como punto de anclaje. │
  └────────────────────┴─────────────────────────────────────────┘

  6.  ¿QUÉ ES LA LECTURA BIÓNICA?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  La Lectura Biónica es una técnica desarrollada por el diseñador
  suizo Renato Casutt.

  La idea es simple: las primeras letras de cada palabra se
  muestran en negrita, y el resto aparece más tenue.

  Ejemplo sin Biónico:
    La lectura es un proceso activo de construcción de sentido.

  Ejemplo con Biónico:
    LA LECtURA ES UN PROceso ACTIvo DE CONSTRUCciÓN DE SENTido.
    (las mayúsculas representan las letras en negrita)

  Al leer, tu cerebro usa esas primeras letras como "ancla" y
  predice automáticamente el resto de la palabra. Esto puede
  reducir el esfuerzo visual y aumentar la velocidad de lectura.

  CÓMO ACTIVARLA:
    Hacé clic en el botón  [ B| BIÓNICO ]  en la barra de
    controles. El botón se iluminará para indicar que está activo.
    Hacé clic de nuevo para desactivarlo.

  ┌─────────────────────────────────────────────────────────────┐
  │  No todas las personas experimentan el mismo beneficio.     │
  │  Algunas sienten que leen más rápido, otras lo usan para    │
  │  mantener el foco. Te recomendamos probarlo unos minutos    │
  │  antes de decidir si te resulta útil.                       │
  └─────────────────────────────────────────────────────────────┘

  7.  CONSEJOS PARA UNA MEJOR EXPERIENCIA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ·  Empezá con el ruido al mínimo y subilo despacio hasta
     encontrar el nivel que te resulte cómodo. Mucho ruido puede
     distraer en lugar de ayudar.

  ·  El modo invertido (fondo claro) suele ser más cómodo con luz
     natural de día. El modo oscuro es más agradable en ambientes
     con poca luz.

  ·  Para sesiones largas, muchas personas encuentran ideal esta
     combinación:

         Biónico ACTIVADO  +  Ruido BAJO  +  Modo INVERTIDO

  ·  Para PDFs con muchas imágenes o tablas complejas, el texto
     puede aparecer en un orden diferente al original. En esos
     casos, puede ser mejor buscar una versión en .txt.

  ·  La aplicación funciona sin internet. Podés guardar el archivo
     noise-reader.html en tu escritorio y abrirlo cuando quieras,
     estés conectado o no.

  8.  SOLUCIÓN DE PROBLEMAS FRECUENTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ──────────────────────────────────────────────────────────────
  El archivo no se abre al hacer doble clic
  ──────────────────────────────────────────────────────────────
  Hacé clic derecho sobre el archivo → "Abrir con" → seleccioná
  tu navegador (Chrome, Firefox, Edge o Safari). Si no aparece
  en la lista, buscá "Elegir otra aplicación".

  ──────────────────────────────────────────────────────────────
  El texto aparece con símbolos extraños (ñ, acentos, etc.)
  ──────────────────────────────────────────────────────────────
  El archivo puede tener una codificación diferente a UTF-8.
  Abrilo en el Bloc de Notas (Windows) o TextEdit (Mac)
  → "Guardar como" → en la opción de codificación elegí UTF-8
  → guardá y volvé a cargarlo en Noise Reader.

  ──────────────────────────────────────────────────────────────
  El PDF no muestra las imágenes o el texto aparece desordenado
  ──────────────────────────────────────────────────────────────
  Algunos PDFs están construidos de formas muy complejas (con
  múltiples columnas, tablas o texto incrustado como imagen).
  Para documentos muy visuales, puede ser mejor buscar una
  versión en .txt del mismo contenido.

  ──────────────────────────────────────────────────────────────
  La aplicación se ve en blanco o no carga
  ──────────────────────────────────────────────────────────────
  Probá actualizando la página con F5 o Ctrl+R. Si el problema
  persiste, intentá abrirla en otro navegador. La aplicación
  funciona mejor en versiones recientes de Chrome o Firefox.

  ──────────────────────────────────────────────────────────────
  El botón "Biónico" aparece oscuro y no se puede hacer clic
  ──────────────────────────────────────────────────────────────
  Esto es normal cuando tenés un PDF cargado en modo de
  visualización clásico. Cargá un .txt para poder usar la
  Lectura Biónica.

  9.  PRIVACIDAD Y SEGURIDAD
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Noise Reader no envía ningún dato a ningún servidor.

  Todo el procesamiento — incluyendo la lectura del archivo, la
  extracción de texto del PDF y la Lectura Biónica — ocurre
  completamente dentro de tu navegador, en tu propia computadora.

  Tus archivos nunca salen de tu dispositivo.

  Podés verificarlo mirando la barra de direcciones del navegador:
  va a mostrar una dirección que empieza con

                          file://

  en lugar de  http://  o  https://  — eso confirma que estás
  trabajando localmente.

  ┌─────────────────────────────────────────────────────────────┐
  │  Si trabajás con documentos confidenciales, esta            │
  │  arquitectura garantiza que nadie más puede acceder a tu    │
  │  contenido. No hay cuentas, no hay servidores, no hay       │
  │  telemetría de ningún tipo.                                 │
